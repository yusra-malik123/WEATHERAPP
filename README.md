# Weather App
This is a simple Weather App that I built to practice working with APIs and JavaScript. It allows users to search for any city and instantly view its current weather conditions in easy way.

## what it can do:
- Search weather by city name
- Display the current temperature
- Show humidity levels
- Display wind speed
- Show the current weather condition with an icon
- Works on both desktop and mobile devices

## Technologies used:
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## Getting Started
If you'd like to run this project on your own computer:

1. Clone this repository.
   ```bash
   git clone https://github.com/yusra-malik123/weather-app.git
   ```

2. Open the project folder.

3. Add your OpenWeatherMap API key in the JavaScript file.

4. Open `index.html` in your browser, and you're ready to go!
